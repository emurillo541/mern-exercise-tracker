A **Single Page Application (SPA)** built with the MERN stack to track exercises completed by a user.  
This project demonstrates full-stack development skills, including building a REST API, working with MongoDB, and creating a responsive React frontend.

---

## Tech Stack
- **Frontend:** React (function components), Vite, React Router, React Icons, CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Tools & Deployment:** npm, GitHub, optionally Vercel/Render

---

## Features
- **Create**, **Read**, **Update**, and **Delete** exercises via a REST API
- Input validation for all exercise fields
- Dynamic HTML table to display exercises
- Delete and edit functionality with React Icons
- Navigation between pages using React Router
- Lifting state / useContext used to manage data flow
- Fully functional SPA with asynchronous fetch API calls
